# ✨ Quote Generator

A beautifully designed Quote Generator built with **React** and **Tailwind CSS**. It generate random quotes and displays them in an aesthetic layout. Users can also copy quotes to their clipboard with a single click.

## 🔧 Tech Stack

- **React**
- **Tailwind CSS**
- **JavaScript (ES6+)**

## 📦 Features

- 🎨 Clean and responsive UI
- 💬 Displays inspirational quotes
- 📋 "Copy to Clipboard" functionality
- 🔄 Generate new quote on button click
